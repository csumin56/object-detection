# 카메라 기반 실시간 사물 인식

YOLOv8 모델을 이용해 카메라에서 실시간으로 사물을 인식하는 프로그램입니다.

## 환경 설정

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
python main.py
```

1. 객체 탐색 시작 여부를 입력합니다 (`y` / `n`)
2. 탐지 시간을 입력합니다 (1~30초)
3. 카메라 화면이 열리고 실시간으로 사물을 인식합니다
4. 시간이 끝나면 가장 많이 탐지된 객체를 출력하고 종료됩니다

```
객체 탐색을 시작하시겠습니까? (y/n): y
몇 초 동안 탐지하시겠습니까? (1~30): 10

10초 동안 탐지를 시작합니다...

객체는 person 입니다.
프로그램 종료
```

- `q`를 누르면 탐지를 중단합니다.
- YOLOv8 모델 파일(`yolov8n.pt`)은 최초 실행 시 자동으로 다운로드됩니다.

## 사용 모델

- **YOLOv8n** (nano): ultralytics에서 제공하는 경량 모델
- COCO 데이터셋 기준 80가지 사물 인식 가능 (사람, 자동차, 의자 등)
